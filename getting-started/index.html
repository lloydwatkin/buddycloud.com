hosted or self-install

buddycloud is a distributed service. Any domain can run a buddycloud server, and each buddycloud server will find, connect and synchronise data with other buddycloud sites. You can use a hosted version of buddycloud, or install you own buddycloud site. We recommend you start with the hosted version and see if buddycloud is right for you.
